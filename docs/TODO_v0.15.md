# 一、麻将语音播报优化（高优先级）✅ 已完成（v0.14.37）

## 当前问题

胡牌播报目前是：

对家
放炮
自己
对对胡

一句一句播报。

整体节奏很慢。

## 修改要求

将多个信息合并成一句。

例如：

对家放炮，自己对对胡。

自己自摸，清一色，四番。

抢杠胡，龙七对。

不要：

自己。
自摸。
清一色。
四番。

要求：

- 减少 speechSynthesis.speak() 调用次数
- 普通打牌播报保持不变（五万、八条、九筒）
- 胡牌一次播报完成
